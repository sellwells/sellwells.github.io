# 结算管理

店铺与平台按照结算单进行结算。

店铺结算单的生成规则是在：店铺详情中进行设置结算日。

每个月按照结算日生成结算单。例如写：1、15，每个月会生成两份结算单。

结算单流程：生成结算单 - 店铺对账 - 平台付款 - 完成。

结算方式为：线下支付。如果需要线上结算，自行去对接分账支付接口即可。
